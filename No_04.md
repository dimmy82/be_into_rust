# rustにハマる巻04

## 1． Enum

#### 普通のやつ
  
```rust
enum Gender {
    Male,
    Female,
}

fn main() {
    let gender = Gender::Male;
    match gender {
        Gender::Male => println!("a man"),
        Gender::Female => println!("a woman"),
    }
    match gender {
        Gender::Male => println!("a man"),
        _ => println!("not a man")
    }
}
```
