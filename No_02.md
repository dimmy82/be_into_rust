# rustにハマる巻02

## 1． Lifetime

  所有権や参照の有効範囲を示す。
  
```#!rust
fn main() {
    let s: String = String::from("hello"); // --------- lifetime 'c -------------|
    //                                                                           |
    { //                                                                         |
        let s1: &str = s.as_str(); // ----------------- lifetime 'b -------|     |
        //                                                                 |     |
        { //                                                               |     |
            let s2: &str = &s[2..4]; // --------------- lifetime 'a -|     |     |
            //                                                       |     |     |
            println!("s1: {}, s2: {}", s1, s2); //                   |     |     |
        } // --------------------------------------------------------|     |     |
    } // ------------------------------------------------------------------|     |
} // ----------------------------------------------------------------------------|
```
